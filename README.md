# Nibble

Classic game written in C++, using ncurses

## Installation

### Debian/Ubuntu Linux
```bash
apt install libncurses5-dev libncursesw5-dev
```

### CentOS/RHEL
```bash
yum -y install ncurses-devel
```

### Makefile
```bash
make
```

## Usage

```bash
./nibble
```
